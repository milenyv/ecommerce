# 🛒 E-commerce com Carrinho (PHP + MySQL)

Projeto de loja virtual completa desenvolvido com **PHP, MySQL, HTML, CSS e JavaScript**, com foco em prática de desenvolvimento full stack.

---

## 🚀 Funcionalidades

* 🛍️ Listagem de produtos dinâmica (via banco de dados)
* 🛒 Carrinho de compras com sessão
* 🔢 Controle de quantidade de itens
* ❌ Remoção de produtos do carrinho
* 🔐 Sistema de login de usuário
* 💳 Finalização de compra
* 📦 Registro de pedidos no banco de dados
* 🎨 Interface moderna e responsiva

---

## 🧠 Tecnologias utilizadas

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
